Help

For security reasons it is not straightforward possible to share the clipboard between your local computer and the client device. However, to take advantage of this convenience, you can use these buttons. The upper one is for copying from the client and the lower one is for pasting into the client clipboard.

Copy from Client

  1. Select text to copy (inside client)
  2. Click on the button
  3. Insert on local machine using the regular key-combination ctrl + v

Copy to Client

  1. Select text to copy (on local machine)
  2. Copy selection using the regular key-combination: ctrl + c
  3. Click on the button
  4. Insert into client's clipboard by pressing ctrl + v
  5. The copied text is now available inside the client machine and can be inserted using regular key-combination
